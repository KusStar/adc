# adc

ADB CLI tool for Android.

## Usage

```bash
npm install -g @kuss/adc

adc
```

Or Just use npx

```bash
npx @kuss/adc
```

```bash
│
◆  adc
│  ● am-start (activity manager start actions)
│  ○ am-stop
│  ○ ime
│  ○ monkey
│  ○ wm
│  ○ exit
```

## Development

```bash
git clone ...

cd adc

# install pnpm first, then
pnpm install

pnpm dev

npm i -g .

# test locally
adc
```
